## Project Overview

Build a Decentralized Personal Data Vault | Own, Store & Monetize Your Data (Web3 Project)

In this tutorial, we build a Decentralized Personal Data Vault that allows users to own, store, control, and monetize their personal data using blockchain technology.

Instead of big tech companies controlling and selling your data, this Web3 application gives full ownership back to the user. Data is encrypted using AES-256 encryption, stored on IPFS decentralized storage, and controlled through Ethereum smart contracts.

This project demonstrates how blockchain can enable secure, privacy-focused data ownership with zero-knowledge guarantees.

## 🔐 Features

- AES-256 encrypted personal data storage
- Decentralized storage using IPFS
- Ethereum smart contracts for permission control
- Share and sell data securely
- Zero-knowledge privacy architecture
- Web3 wallet authentication

## 🧠 What You’ll Learn
